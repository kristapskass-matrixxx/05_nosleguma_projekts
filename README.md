\# 💰 Izdevumu un Ienākumu Uzskaites Sistēma



Šis projekts ir Python programma, kas ļauj uzskaitīt ienākumus un izdevumus, saglabāt tos failā un analizēt finanšu datus.



\---



\## 🚀 Funkcionalitāte



\- Pievienot ienākumus un izdevumus

\- Ierakstīt datumus (YYYY-MM-DD vai YYYYMMDD formātā)

\- Automātiska datuma pārveidošana pareizā formātā

\- Iespēja pievienot komentārus ienākumiem un izdevumiem

\- Izdevumu sadalīšana pa kategorijām

\- Vairāku izdevumu veidu izvēle vienam ierakstam

\- Ierakstu skatīšana un dzēšana

\- Filtrēšana pēc mēneša

\- CSV eksports

\- Statistikas pārskats



\---



\## 🛠️ Kā palaist projektu



1\. Atver termināli mapē ar `app.py`

2\. Palaid komandu:

python app.py





\---



\## 📂 Faili projektā



\- app.py – galvenā programma

\- finanses.json – saglabātie dati

\- finanses.csv – eksportētie dati

\- .gitignore – ignorētie faili
